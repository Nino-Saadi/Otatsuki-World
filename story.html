Le Commencement

L'histoire débute dans le village caché "Le Havre", dix membres otaku se rejoignirent pour
créer l'organisation marchande Otaku parfaite !


Pour se faire ils décident de trouver dans un premier le nom de leurs enseigne.


Et quoi de mieux pour cela que de fusionner une organisation connu dans
le monde otaku : "Akatsuki"; puis y rajouter tout simplement le nom de leurs
passion et mode de vie commun : "Otaku".


S'en suit le nom de l'organisation marchande
la plus reconnu dans le milieu : Otatsuki.


Il fallait ensuite chercher une devise, un objectif
pour pouvoir répandre cette bonne parole :


"Otatsuki, l'univers otaku comme vous l'aimez"
